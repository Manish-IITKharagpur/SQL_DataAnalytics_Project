# 🔍 SQL Exploratory Data Analysis (EDA) & Business Insights

### 📌 Overview
After successfully building the **Medallion Data Warehouse**, this repository focuses on the "Value Creation" phase. Using the **Gold Layer (Star Schema)**, I perform Exploratory Data Analysis (EDA) to answer critical business questions, identify trends, and provide actionable insights for stakeholders.A comprehensive collection of SQL scripts for data exploration, analytics, and reporting. These scripts cover various analyses such as database exploration, measures and metrics, time-based trends, cumulative analytics, segmentation, and more. This repository contains SQL queries designed to help data analysts and BI professionals quickly explore, segment, and analyze data within a relational database. Each script focuses on a specific analytical theme and demonstrates best practices for SQL queries.
